AStyleTransfer

Github Repo: https://github.com/dpinthinker/AStyleTransfer

AStyleTransfer‘s full name is Arbitrary Image Style Transfer. It is an open source android application, which is backend with TensorFlow Lite. It can transfer any image to any other image’s style.

This is an advanced deployment demo of the book, A Concise Handbook of TensorFlow 2. And it is a Android deployment for Google‘s Artistic Style Transfer with TensorFlow Lite. AStyleTransfer is the first android demo for this post.

How to use

It is quite easy to use it.

  • Select one content image, which is the source you want to transfer from.
  • Select one style image, which has the style you want to transfer to.
  • Click the transfer button.

How to get

Github Release:
